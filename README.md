# ✈️ Airline Reservation System

A backend-focused Airline Reservation System developed using **Django** that enables users to search flights, book seats, make payments, generate PDF tickets, and manage bookings securely. The application implements OTP-based authentication, email notifications, and session management to provide a real-world booking experience.

---

## 🚀 Features

### 👤 User Authentication
- User Registration
- Secure Login
- OTP-based Login Verification
- Forgot Password with OTP
- Session Management
- Logout

### ✈️ Flight Booking
- Search flights by source, destination, and travel date
- View available flights
- Check seat availability
- Book single or multiple seats
- Prevent duplicate seat booking

### 💳 Payment
- Multiple payment methods
  - PhonePe
  - Google Pay
  - Amazon Pay
  - Razorpay
  - PayPal
- Booking confirmation page

### 🎫 Ticket Management
- Generate professional PDF airline tickets
- Download tickets
- Automatic ticket delivery via email

### 📧 Email Services
- Login OTP sent to registered email
- Forgot Password OTP
- Ticket sent as PDF attachment

### 📋 Booking Management
- View booking history
- Cancel bookings
- Download booked tickets anytime
